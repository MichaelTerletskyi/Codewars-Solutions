# Instructions

Rock Paper Scissors
Let's play! You have to return which player won! In case of a draw return Draw!.

Examples:

```yaml
rps('scissors','paper') // Player 1 won!
rps('scissors','rock') // Player 2 won!
rps('paper','paper') // Draw!
```
