# Instructions
Write a function called repeatStr which repeats the given string string exactly n times.

```yaml
StringRepeat.repeatStr(6, "I") # "IIIIII"
StringRepeat.repeatStr(5, "Hello") # "HelloHelloHelloHelloHello"
```
