# Instructions
Write a function to split a string and convert it into an array of words. For example:

```yaml
"Robin Singh" ==> ["Robin", "Singh"]

"I love arrays they are my favorite" ==> ["I", "love", "arrays", "they", "are", "my", "favorite"]
```
