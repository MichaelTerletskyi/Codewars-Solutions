```yaml
Level : 8 kyu
```

# Instructions
Create a function that gives a personalized greeting. This function takes two parameters: name and owner.

Use conditionals to return the proper message:

```yaml
case	return
name equals owner	'Hello boss'
otherwise	'Hello guest'
```
