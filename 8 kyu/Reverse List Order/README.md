```yaml
Level : 8 kyu
```

# Instructions

In this kata you will create a function that takes in a list and returns a list with the reverse order.

Examples

```yaml
reverseList([1,2,3,4]) == [4,3,2,1]
reverseList([3,1,5,4]) == [4,5,1,3]
```
